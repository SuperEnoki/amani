Quick Start

Theme Installation

Before you start download the theme and find the .xml demo data in archive.

  1. Login to the WordPress Dashboard
  2. Go to Appearance > Themes
  3. Click "Add New" button
  4. Click "Upload Theme"
  5. Click on the select file / browse and find the directory with theme zip file
  6. Press "Install Now" button
  7. After a successful installation, click on Activate
  8. If you've done everything correctly your theme is installed and you can proceed to Plugins Installation

Notice: If you have an error during installation like this:

Are you sure you want to do this?

It is probably that you're trying to use the full theme archive instead of just a WordPress theme. Try to unzip this archive and find the theme file inside

If you're still having issues try installing theme manually using FTP client

Recommended Plugins Installation

  1. Go to Appearance > Install Plugins
  2. Find "Instagram Widget" and click "Install"
  3. Next you'll see text "Plugin activated successfully." click "Return to Required Plugins Installer"
  4. Repeat the steps if you need any other plugins

Quick Start

  1. Before you start: Follow the previous steps 1.1 & 1.2
  2. Download "Demo Data" from Marketplace and unzip it, you will see the .xml files
  3. Select the xml file with the name of chosen style set demo
  4. Go to Tools > Import
  5. Select WordPress from the list, it will move you to Importer Plugin itself or installation page
  6. Click "Install Now" if you haven't installed this plugin yet
  7. After WordPress Importer Plugin installation click on Activate Plugin & Run Importer link
  8. Select chosen demo .xml file from your computer
  9. Click on button "Upload file and Import"
  10. In "Assign authors" you can either add new user or select existing WordPress author to add default posts and pages to and enable checkbox - Download and import file attachments if you want to import images from demo (in case they are available)
  11. After you've assigned authors click "Submit" button and demo content without images should be imported
  12. Now add Featured Images to each post,they are not included with theme because they are licensed only for one domain, you can choose your own images. Posts - Edit - Featured Image - Upload / Select image. Each post should have featured image for the best experience.
  13. Featured Users information - go to Users > All Users > Edit user you want to feature. Change the user role to at least Author, imported users are Subscribers and can't be featured. Add the "Biographical Info", "Instagram Username" and Social Networks links. Do the same for the second/third/fourth author you want to feature.
  14. Featuring User - Go to Appearance > Customize > Header > Featured Users and select the featured user. You can use Command/Ctrl + mouse click to select multiple users.
  15. Top Featured Posts - Appearance > Customize > Content > Top Featured Category select the existing category you want to be featured. You can add new category in Posts > Categories. Make sure you have featured images set in posts for best experience.
  16. Bottom "Editor's Picks" Posts - Appearance > Customize > Content > Bottom Featured Category select the existing category you want to be featured. You can add new category in Posts > Categories. Make sure you have featured images set in posts for best experience.
  17. Widgets - Navigate to Appearance > Widgets. Move (drag-and-drop) widgets "Thumbnail Category Posts" and "Travel Recent Posts" from Available Widgets to Sidebar area, you can add widget title and select the category you want. Click "Save" on each widget after you've done editing
  18. Footer menu - Go to Appearance > Menus - Add new menu, select the items and add them to menu and check the "Footer Navigation" Display location. Click "Save Menu".
  19. Footer Instagram - Go to Appearance > Customize > Footer > "Instagram Username" and enter your instagram username (e.g. doyoutravel)
  20. You've done customizing main style. Now you can change the style set go to Appearance > Customize > Content > Style Set

Advanced Customization

Every theme styling option and setting is located in theme customizer. Navigate to Appearance > Customize and try out all settings

After you're done with editing click "Save & Publish". If you won't do this changes won't be saved.

Hint: Before you start - look at all available and popular fonts here.

Site Identity

  • "Site Title" - Change the title of your website if you need
  • "Tagline" - Describe what is your website or blog about
  • "Icon" - You can upload custom favicon. It will be displayed on the left side of the browser tab

Header

  • "Logo Font" - Select font for logo if you did not upload custom logo image
  • "Logo Font Size" - Size of logo font (in pixels, optimal value: 48px)
  • "Uppercase" - Check if you want to use all capital letters in logo
  • "Logo Font Weight" - Font weight of logo. Make sure selected font supports weight you're planning to use
  • "Italic" - Make cursive logo font
  • "Show Tagline" - Check if you want to show your website tagline under logo
  • "Tagline" - Enter tagline text you want to feature under logo
  • "Logo Image" - Upload custom logo image (use .png image with x2 size for retina displays) or leave it blank for text logo
  • "Retina Logo" - Check this option if you've uploaded the retina logo image (x2 size of original logo)
  • "Featured Users" - Select users you want to feature in header (minimum required role - author)
  • "Instagram Username" - Username for the Instagram feed in header e.g. "doyoutravel"

Content

  • "Accent Color Scheme" - Select main theme color from one of the available schemes
  • "Accent Color" - Custom main theme accent color
  • "Secondary Color" - Custom secondary theme color
  • "Headings Font" - Select font for headings
  • "Uppercase" - Check if you want to capitalize letters in headings
  • "Headings Font Weight" - Font Weight of Headings. Make sure selected font supports weight you're planning to use
  • "Italic" - Make cursive headings font
  • "Post Title Font Size" - Font Size of Posts Title (in pixels, optimal value: 48px)
  • "Body Font" - Select font for basic website text - paragraphs, links, etc.
  • "Body Gradient" - Select solid color or the direction of gradient (taken from Accent color and Secondary color)
  • "Body Background" - Color for solid Background
  • "Body Background Theme" - If you have a light background or gradient it's better to change this to "White", in case you have dark body background or gradient switch it to "Dark"
  • "Content Background" - Background for content area (post list, categories, singular posts and pages)
  • "Content Background Theme" - Switch this to "Dark" if you prefer black or dark background of content
  • "Top Featured Layout" - Select the layout for Top Featured posts
  • "Sidebar Position" - Set the position of sidebar or hide it completely
  • "Blog Layout" - Select the layout of blog posts in categories, archives and on home page
  • "Show Social Share Under Post" - Show the "Share" block under the singular post (requires Bekento Tools plugin)
  • "Infinite Scroll" - Hide pages pagination and enable infinite scroll
  • "Top Featured Category" - Set the Top Featured posts category
  • "Bottom Featured Category" - Set the Bottom Featured posts category
  • "Exclude Featured Categories from Home Page" - Exclude posts that are featured from home page
  • "Hide Featured Categories from Meta" - Don't display the featured category name in meta

Footer

  • "Featured Users" - Select users you want to feature in footer
  • "Copyright Text" - Custom footer copyright text
  • "Instagram Username" - Username for the Instagram feed in footer e.g. "doyoutravel"

Font Charsets

https://www.instagram.com/doyoutravel/

Menus

  1. Click "Add Menu"
  2. Enter new menu name in appeared box
  3. Click "Create Menu"
  4. Click "Add Items"
  5. Mark every page, post or other item that will be in the menu
  6. After editing menu you can rearrange items by dragging and dropping them or clicking "Reorder" button, you can also create sub-menus
  7. In "Menu locations" mark desired location of the created menu

Widgets

  1. Select the Widget Area you want to add Widgets to
  2. Click "Add a Widget"
  3. Select a Widget you want to add
  4. After a Widget is added you can customize it
  5. Reorder Widgets by dragging and dropping them or clicking "Reorder" button

Plugins and Widgets

If you did not install plugins for widgets please follow step #1.2

After you have all recommended plugins installed navigate to Appearance > Widgets

Twitter Widget

  1. Add "Twitter" Widget to Widget Area
  2. "Title" - Enter the title of Widget, you can enter anything or leave blank
  3. "Number of Tweets" - Select number of displayed tweets
  4. "Time in Minutes between updates" - select the time between updates, use 5 minutes for best experience
  5. Go to https://apps.twitter.com/ and sign in (you should have Twitter account, if not - register).
  6. Center of the screen click "Create a New Application."
  7. Enter a unique name (anything you want), a description (again this is just for you), and your site's URL. You can leave the Callback URL empty as it is not used for this implementation.
  8. You will be taken to a new screen, there's one more step then you can copy all the details into correct fields of the Widget and be on your way.
  9. OK, click the "Create my Access Token" button. This is a shortcut to authenticate your own account with your application (so you never need use your password).
  10. Click the Details Tab as all the information you need is presented hereso you can just copy the required information into the exact corresponding inputs fields of the Widget "Twitter API Settings".
  11. Click "Save"

Instagram Widget

  1. Add "Instagram" Widget to Widget Area
  2. "Title" - Enter the title of Widget, you can enter anything or leave blank
  3. "Username" - Enter your Instagram account username
  4. "Number of photos" - Enter any even value for best experience (2, 4, 6, etc.)
  5. "Photo size" - you can select photo size for embed Instagram images, use "Large" for best experience
  6. "Open links in" - you can select "New Window" so all links to Instagram will be opened in new tabs
  7. "Link text" - not used in theme, leave it blank
  8. Click "Save"

Profile Widget

  1. Add "Profile" Widget to Widget Area
  2. "Title" - Enter the title of Widget, you can enter anything or leave blank
  3. "User" - Select user you want to feature
  4. Click "Save"

Contact Form 7

Open the Contact > Contact Forms menu on your WordPress administration panel. You can manage multiple contact forms there.

Just after installing the Contact Form 7 plugin, you'll see a default form named "Contact form 1", and a code like this:

[contact-form-7 id="1234" title="Contact form 1"]

Copy this code. Then, open the edit menu of the page Page > Edit into which you wish to place the contact form. A popular practice is creating a page named "Contact" for the contact form page. Paste the code you copied into the contents of the page.

Now your contact form setup is complete. Visitors to your site can now find the form and start submitting messages to you.

Info on advanced Contact Form configuration is located at official Conatct Form 7 page

  9. Click "Save"

Translation (localization)

  1. Login to WordPress Dashboard and Navigate to Plugins > Add New
  2. In the search field enter "Loco Translate"

  3. In the box of "Loco Translate" plugin click "Install Now"
  4. Wait for the text "Successfully installed the plugin Loco Translate" to appear
  5. Click "Activate Plugin"
  6. Navigate to Loco Translate > Themes
  7. Find the theme you want to translate, click it

  8. Click on a link "Create Template"
  9. Click blue button "Create Template"

  10. After it you'll see message "OK: Template file created", click "New Language"

  11. Select language you want theme to be translated to and click "Start Translating"
  12. Translate everything you want (You can use searchbar to translate particular text)
  13. After you translate every needed string click "Save"

  14. Congratulations! Your website should be translated now!

FAQ

What featured image size to use

Recommended using image with dimension 16:9 or 16:10 with a size over 1920x1080 pixels or higher. This should make your featured image look great both on retina, mobile/tablet and standard displays :)

Can't change fonts

Some fonts don't have all font weights available. Go to Apperance > Customize > Header > Logo Font Weight and switch the font weight to "400 Regular" and see if this helps. Do this for each font you are using. To see what Font Weight can be selected for each font go to fonts.google.com find the font you want to use, hover the font and click on second dropdown.

How to add posts to top featured like in demo

Appearance > Customize > Content > Top Featured Category select the existing category you want to be featured. You can add new category in Posts > Categories. Make sure you have featured images set in posts for best experience

Instagram Feed is not updating

Check if you're using caching plugin and if you have one clear the website cache. The cache of the Instagram Widget is clearing every hour, if you want to update it manually you can use this plugin: artiss-transient-cleaner It will remove all cached transient values. Also check if Instagram is not down at Downdetector

I'm having issue with Internet Explorer 10

Unfortunately, theme does not support older browser versions

GDPR / Data Protection

Please read this section carefully. Violating Data Protection / GDPR rules could result in fines depending on country.

Theme does not collect any data and does not store any visitor information, selecting and using Google Fonts could be a violation of GDPR. Some plugins bundled in theme could violate GDPR rules.

Theme uses remote connection to Google servers if Google Font is selected, this could result in GDPR violation. The solution is to select Helvetica / Georgia font that will be used from local file available both on Mac, Windows, Android and Linux. Theme will use default OS font as a fallback.

If you're using Instagram widget there will be a remote connection to Instagram servers, this could be potentially violation of GDPR rules for users. The solution is to disable Instagram plugin.

Twitter widget will connect to twitter servers, it could be potentially violation of GDPR rules for users. The solution is to disable Simple Twitter Tweets plugin.

Post Views Counter. This plugin does not collect any personalized information, but to fully comply with european GDPR rules it is recommended to disable it.

Contact Form 7. Disabling this plugin is recommended.

Disable all other third-party plugins, you can't be 100% sure about their GDPR compliance and make sure you are not using Google Analytics, Google Ads and any other Google / Facebook, etc. service, they potentially could have loopholes and could violate GDPR.
